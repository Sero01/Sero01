# Hey, I'm Parvez 👋

I build **local-first AI agents** and **self-hosted tools** — software that runs on your own machine, owns its own data, and doesn't phone home.

## 🤖 Agents

- **[Emma](https://github.com/Sero01/Emma)** — autonomous personal agent with a plan-first loop (PLAN → EXECUTE → SYNTHESIZE), a skill registry, sub-agents, budget caps, and a Discord interface
- **[Norman](https://github.com/Sero01/Norman)** — lean, fully-offline agent on Ollama; "sniper agent" philosophy — minimal context, single purpose, no bloat
- **[Speculum](https://github.com/Sero01/speculum)** — introspection dashboard that visualizes how an agent actually works: sessions, context-window health, topic landscape
- **[memory-wiki](https://github.com/Sero01/memory-wiki)** — turns an agent's session history into a searchable, self-hosted knowledge base

## 🛠️ Tools & apps

- **[watchparty](https://github.com/Sero01/watchparty)** — stream local videos to friends' phones in sync, with built-in voice chat
- **[inkling](https://github.com/Sero01/inkling)** — a journal that reads *how* you write: punctuation, casing, rhythm, tone
- **[WhispnoteAI](https://github.com/Sero01/WhispnoteAI)** — voice-first note-taking app (React Native / Expo)
- **[FinExtract](https://github.com/Sero01/finextract)** — browser extension that turns bank-statement PDFs into clean Excel/CSV
